## Use

Clone, install dependencies, build, and connect it to an MCP-compatible client.

~~~bash
git clone https://github.com/ResearchForumOnline/mcp-titan-cognative-memory.git
cd mcp-titan-cognative-memory
npm install
npm run build
~~~
